# Replicating-LIDAR-Based-Aboveground-Biomass-Regression-using-Aerial-Images
This is the code associated with the paper 'Replicating LIDAR-Based Aboveground Biomass Regression using Aerial Images'

The code used in the paper is located in patch_regression.ipynb. In order to run the code, one must download the AGBD data from https://daac.ornl.gov/cgi-bin/dsviewer.pl?ds_id=2443 and place it in a folder within this repository.

The cells of the notebook must be run in order, from start to finish, to ensure that the right files are created before the models are trained.
