# Replicating-LIDAR-Based-Aboveground-Biomass-Regression-using-Aerial-Images
This is the code associated with the paper 'Replicating LIDAR-Based Aboveground Biomass Regression using Aerial Images'

The code used in the paper is located in patch_regression.ipynb. All the data produced by each cell is located in the repository. Many of the files are old and not used in the notebook, but are kept in the repository for completeness.
